Ex01: Display process of a static page.

When a user requests a static page in Django:

  1. The browser sends an HTTP request to Django’s server.
  2. The URL dispatcher matches the request path to a defined path() in urls.py.
  3. The matched view function in views.py executes.
  4. The view calls render(request, 'template.html') to generate HTML from the template.
  5. Django’s template engine reads the file, replaces context variables, and returns an HttpResponse.
  6. The response is sent back to the browser and displayed to the user.
