What is Markdown?

Markdown is a lightweight plain-text markup language for writing formatted documents. It was designed to be readable as plain text and easily converted to HTML. Variants like CommonMark and GitHub Flavored Markdown (GFM) add more features.

Basic syntax

Headings

# Heading 1
## Heading 2
### Heading 3

Paragraphs & line breaks

Separate paragraphs with a blank line. To force a line break inside a paragraph, end a line with two spaces.

This is a paragraph.  
A new line in the same paragraph.

Emphasis

*italic*     _italic_
**bold**     __bold__
***bold+italic***

Lists

Unordered: use -, *, or +. Indent for nested lists.

- Item 1
- Item 2
  - Nested item

Ordered: numbers + dot. Many renderers auto-number correctly even if you write 1. for each item.

1. First
2. Second
3. Third

Blockquotes

> This is a blockquote.
> It can wrap multiple lines.

Code

Inline code: wrap with backticks.

Use the `print()` function.

Fenced code block: use triple backticks and optionally specify the language for highlighting.

```python
def hello():
    print("Hello")
```

Links & Images

[Link text](https://example.com "optional title")
![Alt text](https://example.com/image.jpg "optional title")

Inline HTML

You can use raw HTML tags when Markdown doesn't offer the feature you need. Keep a blank line before and after block-level HTML.

<p>Hidden content</p>
</details>

Best practices & tips

  • Keep your Markdown readable as plain text — that's the main advantage.
  • Put blank lines around big elements (lists, code blocks, tables) for consistent rendering.
  • Use fenced code blocks and specify the language for syntax highlighting.
  • Prefer descriptive link text over "click here."
  • Test in the target platform (GitHub, Notion, MkDocs, etc.) because features vary.

Compatibility & variants

Common variants:

  • Original Markdown — minimal feature set.
  • CommonMark — a formal spec for consistent behavior.
  • GitHub Flavored Markdown (GFM) — adds tables, task lists, autolinks, etc.
  • MultiMarkdown, Markdown Extra — add footnotes, definition lists and other features.
